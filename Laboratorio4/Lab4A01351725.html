Sección de Preguntas

1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Java es un lenguaje orientado a objetos mayormente de servidor que ejecuta código en navegador a través de máquinas virtuales y Javascript es un lenguaje basado en prototipos, nativo de navergadores, de código de cliente, con mayor versatilidad y menos pesado.

2. ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

  • Date.now()=Devuelve el valor numérico de la fecha actual.
  • Date.parse()=Transforma una cadena representante de una fecha y la devuelve en milisegundos.
  • Date.getMilliseconds()=Devuelve los milisegundos entre 0 y 9999.
  • Date.getDate()=Devuelve el día del mes entre 1 y 31.
  • Date.getFullYear()=Devuelve los cuatro dígitos del año.

3. ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

  • filter()=Filtra el arreglo y regresa otro arreglo filtrado con la condición buscada.
  • find()=Recorre el arreglo y regresa la primera coincidencia encontrada con la búsqueda.
  • sort()= Ordena los elementos del arreglo y retorna el arreglo ordenado.
  • concat()=Une dos o más arreglos.
  • push()=Añade elementos al final del arreglo.

5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Las variables globales es de acceso total en el código. Esto significa el código puede tomar esta variable a su conveniencia en cualquier parte de este, lo que implica su uso dentro y fuera de las funciones. Un mal o indebido uso de estas variables puede crear dependencias y movimientos desconocidos que pueden arruinar el programa.
